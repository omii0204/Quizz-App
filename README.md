# Quizz-App
Here's a strong resume description for this project:

Master Quiz – Web-Based Quiz Application
HTML, CSS, JavaScript, Java, Spring Boot, REST API

Developed a full-stack interactive quiz web application featuring multiple programming language quizzes (Java, Python, JavaScript, .NET) with a dynamic landing page, navigation, and quiz package selection.
Built a responsive frontend using HTML5, CSS3, and vanilla JavaScript with smooth scroll navigation, animated image cards, and package-based quiz selection.
Integrated a Spring Boot REST API backend (/api/questions) to dynamically fetch quiz questions, enabling real-time question loading and answer validation.
Implemented a timed quiz engine with auto-navigation between questions, score tracking, and a result page displaying performance feedback based on percentage scored.
Used localStorage for client-side score persistence across page redirects and managed application state without any external frontend framework.
Designed a multi-page architecture (index.html → java-quiz.html → result.html) with modular CSS and JavaScript files for maintainability.
